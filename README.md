# Next.js + TypeScript

## Homework 1

Create a side structure for displaying users, posts, comments from Jasonplaceholder.
/users – display a list of users with an active link
/posts – display a list of posts with an active link
/comments – display a list of comments with an active link
/users/id – display the current user
/posts/id – display the song post
/comments/id – display the current comment

## Technologies
- Next.js (App Router)
- Route Handlers (API routes)
- Fetch API
- TypeScript
- TailwindCSS

## Getting started

1) Clone the repository:
```bash
git clone <repository-url>
```
2) Go to the project folder:
```bash
cd <project-folder>
```
3) Install dependencies:
```bash 
npm install
```
4) Run project:
```bash
npm run dev
```
The project will be available at: http://localhost:3000
