# Next.js + TypeScript

## Homework 3

Create a page that displays all cars (with minimal information) using API http://owu.linkpc.net/carsAPI/v1/doc.
Create a page with a form for creating car objects.
Create services for working with the API.
Add form validation using Joi and react-hook-form

## Technologies
- Next.js (App Router)
- Route Handlers (API routes)
- Fetch API
- React-hook-form
- Joi
- TypeScript
- TailwindCSS

## Getting started

1) Clone the repository:
```bash
git clone <repository-url>
```
2) Go to the project folder:
```bash
cd <project-folder>
```
3) Install dependencies:
```bash
npm install
```
4) Run project:
```bash
npm run dev
```
The project will be available at: http://localhost:3000
