# Next.js + TypeScript

## Homework 2

Create a page that displays all cars (with minimal information) using API http://owu.linkpc.net/carsAPI/v1/doc.
Create a page with a form for creating car objects.
Create services for working with the API.
Data fetching is performed via services (fetch API).
Use server actions for sending requests to the external API.

## Technologies
- Next.js (App Router)
- TypeScript
- TailwindCSS
- Fetch API
- Server Actions
- Zod server validation

## Getting started

1) Clone the repository:
```bash
git clone <repository-url>
```
2) Go to the project folder:
```bash
cd <project-folder>
```
3) Install dependencies:
```bash
npm install
```
4) Run project:
```bash
npm run dev
```
The project will be available at: http://localhost:3000
